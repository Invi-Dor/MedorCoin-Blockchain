Careers at MedorCoin Ecosystem

The MedorCoin ecosystem combines blockchain infrastructure, crypto APIs, trading education, token liquidity services, and security focused development. Our mission is to create opportunities for talented professionals around the world to participate in building the next generation of decentralised financial technology.

Blockchain Developer (Contract Engineer – Remote)

As a blockchain developer you will help build the technical backbone of the ecosystem. Your work will involve improving network performance, developing smart contract integrations, and supporting secure blockchain infrastructure.

Duties

  • Develop blockchain architecture
  • Implement smart contract integrations
  • Improve node performance
  • Collaborate with API developers
  • Maintain blockchain security standards

API Integration Specialist (Hybrid)

This role focuses on connecting external applications with our crypto APIs, enabling developers, traders, and data platforms to interact with blockchain data efficiently.

  • Integrate APIs into partner platforms
  • Test endpoint functionality
  • Assist developers using our APIs
  • Improve API documentation

Crypto Wallet Developer (Remote)

Design and maintain secure wallet systems that allow users to store, transfer, and interact with digital assets within the ecosystem.

  • Develop wallet architecture
  • Implement encryption and security layers
  • Ensure compatibility with blockchain nodes
  • Improve wallet usability

Frequently Asked Questions

How do I apply for a job?
Click the “Apply Now” button under the position. Applications will be available soon as the recruitment portal is being prepared.
Can I work remotely?
Yes. Some roles are remote while others are hybrid or on site depending on the responsibility.
Do I need blockchain experience?
For technical roles such as blockchain development or security analysis, blockchain experience is highly recommended.
What industries does the ecosystem operate in?
Blockchain infrastructure, crypto APIs, liquidity services, digital asset security, and crypto education.
