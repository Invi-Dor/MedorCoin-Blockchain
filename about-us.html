About MedorCoin

MedorCoin, ticker MEDUR, is a security-focused blockchain designed for clarity, auditability and long-term sustainability. It combines simple proof-of-work mining, structured transaction processing, deterministic fee mechanics and transparent on-chain accounting.

What began as a compact and auditable blockchain protocol has evolved into a complete ecosystem. MedorCoin now provides a unified platform supporting miners, developers, investors, learners and liquidity participants. All value flows, reward splits, treasury allocations and protocol fees remain fully on-chain and verifiable.

Core Blockchain Foundation

MedorCoin ensures:

  • Structured transaction validation with strict signature, nonce, balance and gas checks
  • Fee-based transaction prioritisation within a thread-safe mempool
  • Deterministic reward distribution between miners and the protocol treasury
  • Clean key-value state model for smart contract code and storage
  • Configurable base fee and protocol fee policies
  • Simple and transparent mining logic designed for auditability

Mining rewards and transaction fees are automatically split between miners and the MedorCoin Treasury. Staking and liquidity locking operations apply small protocol fees credited directly to the treasury. The platform is designed for simplicity and security, reducing attack surface and improving transparency.

Developer Infrastructure and Crypto APIs

MedorCoin provides developers with access to unlimited API endpoints to integrate wallets, trading interfaces, analytics systems and external services directly with the blockchain. The crypto APIs enable:

  • Programmatic transaction submission
  • Real-time balance and state queries
  • Mempool inspection
  • Treasury tracking and reward accounting
  • Configurable protocol parameters

These APIs make it possible to build applications and services without artificial limits, making MedorCoin a complete developer ecosystem.

Liquidity and Token Locks

MedorCoin allows users to securely lock liquidity or tokens for small protocol fees. The platform provides:

  • Token locking
  • Liquidity locking
  • Transparent fee allocation to the treasury

All locking operations are fully auditable and remain on-chain, enhancing both security and transparency.
