Frequently Asked Questions

1. How do I get an API key?

You can generate an API key from your account dashboard. Keep it secure and use it to authenticate your requests.

2. Are there limits on API requests?

Yes, we have rate limits to maintain network stability. Limits depend on your subscription plan.

3. Can I use the API for trading bots?

Yes, the API supports automated trading for spot, futures, and options markets within your permissions.
