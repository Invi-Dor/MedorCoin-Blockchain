MedorCoin Documentation

Tip: Keep private values out of your repo. Use a gitignored file or environment variables.

Quick Start

  1. Clone the repo: git clone https://github.com/invi-dor/MedorCoin-Blockchain
  2. Open index.html locally to view the landing page.
  3. Edit docs.html as your documentation grows.

Configuration

Recommended public and private configuration files:

{
  "name": "MedorCoin",
  "ticker": "MEDUR",
  "stakeLockFeeBps": 50,
  "liqLockFeeBps": 50,
  "miningRewards": "enabled"
}
{
  "minerRewardBps": 9000,
  "protocolRewardBps": 1000,
  "baseFeePolicy": "split",
  "splitBpsToTreasury": 7000,
  "splitBpsToMiners": 3000,
  "protocolTreasuryAddress": "0x85708d61FEfcbb6eb13C72b0D42bCeB246F06dd0"
}

Environment variables (optional)

# .env (example)
MINER_REWARD_BPS=9000
PROTOCOL_REWARD_BPS=1000
BASE_FEE_POLICY=split
SPLIT_BPS_TREASURY=7000
SPLIT_BPS_MINERS=3000
PROTOCOL_TREASURY_ADDRESS=0x85708d61FEfcbb6eb13C72b0D42bCeB246F06dd0

Deployment

  1. Commit index.html and docs.html to your gh-pages branch or main with Pages enabled.
  2. Set Pages to deploy from / root. URL will be your GitHub Pages link.
  3. Verify navigation links work between Home, Docs, and Explorer.

FAQ

Q: Can I change the accent color?
A: Yes, update --accent in the :root CSS.

Q: Where do I add new sections?
A: Duplicate the article blocks or add new headings and link them in the TOC.
